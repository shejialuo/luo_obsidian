# Environment Setup

## Installation

### Archlinux

```sh
sudo pacman -S beancount fava
```

### Macos

```sh
brew install beancount fava
```

## Editor Configuration

Use `VsCode` as the editor, install the following two plugins:

+ [Beancount](https://github.com/Lencerf/vscode-beancount)
+ [Beancount Formatter](https://github.com/dongfg/vscode-beancount-formatter.git)
