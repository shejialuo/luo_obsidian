# 李大霄投资战略

## 第一部分

这一部分李大霄主要是提出的自己的一些基础投资认知：

+ 普通人要远离杠杆，能把自己的财富管理好就不错了。
+ 投资之前要确定目标：力求达到利润指标及可能承受的风险（损失率），准备动用的资金量为多少。其次要
思考投资的流程，基于财力状况和投资时间做决策。
+ 普通人应选择投资策略而不是投机策略。
+ 应该选择“坐轿子”而不是“抬轿子”，选择低位购买优质股票。
+ 交易的最高境界就是少交易。
+ 要在股票市场赚钱，一定要有概率意识，树立在股票学习的班上考前5名的意识。

## 第二部分

股市有四个风向标：

1. 上市公司买卖股票的时机：上市公司卖股票就是股价涨到比较满意的时候。买股票的时候，就是股价跌到它
比较满意的时候，这个时候就是开始买股票的时候。
2. 中石油增持股票。
3. 汇金公司的出手是市场稳定的重要条件。
4. 社保基金入市通常在市场的底部，离场可能就是市场的顶部。

## 第三部分

李大霄老师首先讲述其对“讲与听”的看法：

+ 对“说”的人研究一下：研究其过往，判断是否始终言行一致。
+ 对“听”的人研究一下：分析自己，到底我是想听别人的观点，还是只听自己的观点？

其次是对舆论的理解：

+ 尤其要注意股市中的群体效应，把控风险。
+ 舆论“一边倒”的情况需要更加警惕。

进而是一些情感处理和技巧：

+ 要忍受住不被认同的孤独。
+ 股市投资要经得起折腾。
+ 避免追涨杀跌。
+ 获利最明显的时候需要淡泊，市场亏损最严重时需要勇气。**股票市场并不是人生的全部**。如果这个人
除了钱以外什么也没有，这就不是完美的人生，钱只是人生的一部分，不能放在最重要的位置。这样才可以把
理财的人生观摆正。

## 第四部分

给出了一些对于A股市场的理解（看空方面）：

+ 游戏规则决定投资者的行为，交易机制决定了做多才能盈利的思维方式。
+ 投资者的习惯决定了判断氛围，其实是人性的弱点使然。
+ 商业模式决定观点取向，大部分商业机构并没有商业计划。
+ 大部分的观点发出这都是受雇于商业机构，独立的研究机构很难找到生存的空间。
+ 看空一只股票更明显，因为看空在A股并不创造效益。

给出了对股指期货的意见：

+ **普通人别做融资融券和股指期货**。
+ **这个领域仅仅是专业人士的专利**。
